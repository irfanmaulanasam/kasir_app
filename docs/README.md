# Kasir App

Simple offline-first POS application built with Flutter.

Designed for:
- Small shops
- Grocery stores
- Traditional retailers
- Small wholesalers

Focused on:
- Fast transactions
- Simple inventory management
- Low-end Android devices
- Offline operation

## Current Status

Current development target:

- BASIC version stabilization
- Inventory reliability
- Transaction workflow refinement
- UX simplification
