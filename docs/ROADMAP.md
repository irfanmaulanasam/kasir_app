# ROADMAP

## BASIC — Operasional Harian

Target:

Pemilik usaha dapat menjalankan kegiatan harian dan memahami kondisi usaha tanpa perlu belajar sistem yang rumit.

### Produk

* Manajemen produk
* Harga jual
* Harga beli
* Minimum stok

### Stok

* Tambah stok
* Kurangi stok
* Riwayat stok
* Warning minimum stok

### Penjualan

* Transaksi penjualan
* Pembayaran tunai
* Pembayaran transfer
* Pembayaran QRIS
* Nota transaksi
* Riwayat transaksi

### Piutang

* Penjualan tempo
* Daftar piutang
* Pembayaran piutang
* Riwayat pembayaran piutang

### Pengeluaran

* Catat pengeluaran
* Riwayat pengeluaran

### Arus Kas

* Kas awal
* Ringkasan kas
* Aktivitas keuangan

### Keamanan Data

* Backup database
* Restore database

### Pengaturan Toko

* Nama toko
* Alamat
* Nomor telepon
* Footer nota

## ADVANCED — Pengelolaan Usaha

Target:

Pemilik usaha mulai memahami hubungan antara penjualan, pembelian, hutang, dan keuntungan usaha.

### Supplier

* Data supplier
* Hutang supplier
* Pembayaran hutang supplier
* Riwayat pembayaran hutang

### Pembelian

* Pembelian stok
* Supplier pada pembelian
* Pembelian tempo
* Riwayat pembelian

### Pelanggan

* Data pelanggan
* Histori transaksi pelanggan

### Konsinyasi

* Barang titipan
* Supplier konsinyasi
* Perhitungan bagi hasil

### Laporan

* Ringkasan penjualan
* Ringkasan pengeluaran
* Ringkasan piutang
* Ringkasan hutang
* Laporan laba rugi sederhana

### Multi User

* Hak akses
* Aktivitas pengguna

### Sinkronisasi

* Cloud sync
* Backup cloud
* Restore cloud

### Multi Cabang

* Banyak toko
* Sinkronisasi data
* Monitoring cabang

### Dashboard Owner

* Ringkasan bisnis
* Performa toko
* Monitoring operasional

### Analitik

* Produk terlaris
* Tren penjualan
* Tren pelanggan
